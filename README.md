# 坦克大战

电脑键盘版网页小游戏，基于 Phaser 3.90、JavaScript ESM 和 esbuild 制作。项目最终打包为单个离线 HTML 文件，直接用浏览器打开即可游玩。

## 在线/本地游玩

直接打开仓库中的：

```text
坦克大战.html
```

如果从源码重新构建：

```bash
npm install
npm run build
```

构建后会重新生成 `坦克大战.html`。

## 操作方式

| 操作 | 按键 |
|---|---|
| 移动 | `WASD` 或方向键 |
| 开火 | `空格` 或 `J` |
| 暂停 / 继续 | `P` 或 `Esc` |
| 菜单按钮 | 鼠标点击 |

## 游戏规则

- 玩家驾驶坦克守护地图下方基地。
- 消灭当前关卡全部敌人即可通关。
- 玩家拥有 2 条生命，中弹后会复活并获得短暂无敌。
- 复活无敌期间会显示蓝色保护盾光环，光环消失后恢复正常受击。
- 基地被敌方子弹、玩家子弹或其他伤害摧毁时失败。
- 坦克身体接触只会碰撞阻挡，不会直接失败。

## 内容特性

- 5 个单人关卡：边境初战、水草迷阵、寒钢要塞、终极防线、钢铁风暴。
- 敌人类型包括普通、快速、火力和装甲。
- 地形包括砖墙、钢墙、水面、草丛、冰面和基地。
- 道具包括火力升级、护盾、冻结、炸弹和基地钢化。
- 支持本地最高分与关卡解锁存档。
- 精致复古像素风，程序化生成纹理和音效。

## 开发命令

```bash

npm run build
```

测试覆盖地图解析、碰撞、伤害、道具、存档、计分、关卡解锁、构建产物等核心逻辑。
