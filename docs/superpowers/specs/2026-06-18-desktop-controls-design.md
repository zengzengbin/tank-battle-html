# 坦克大战电脑键盘版设计

## 目标

将游戏收敛为纯电脑网页版本，删除手机触控和旋转提示，使用键盘完成战斗操作，并消除出生点砖墙造成的“子弹只能飞一个身位”误解。

## 设计

- 战斗画面保持 960×540，移除左右触控区，将 13×13 战场放大并居中偏左，右侧显示键位和战况。
- `WASD` 或方向键移动，`空格` 或 `J` 开火，`P` 或 `Esc` 暂停；鼠标仅用于菜单按钮。
- 四关玩家出生点上方均保持开放，玩家可直接进入战场并向上远射。
- 子弹沿当前方向飞行，直到碰到可阻挡物、坦克、基地或地图边界；不穿透砖墙和钢墙。
- 最终 HTML 删除触摸专用文案、虚拟方向键、触屏开火键、竖屏遮罩和移动端媒体查询。

## 验收

- 自动测试验证四张地图的出生点与上方通道为空。
- 构建测试验证最终 HTML 不含手机、横屏、虚拟触控提示，并包含 `空格 / J` 键位文案。
- 浏览器实测开始游戏、键盘移动、连续开火、暂停，并确认开阔区域中的子弹跨越多个格子。
